# gym-promotional-website
A responsive, modern promotional website for a fitness center, built for a frontend developer internship evaluation.

## Project Name
Gym - Transform Your Life

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)

## Setup Instructions
1. Clone this repository to your local machine.
2. Navigate to the `gym-website` directory.
3. Open the `index.html` file in your preferred web browser. No compilation or build step is required.
   
   Alternatively, you can run a local development server using tools like VS Code Live Server to view the site:
   ```bash
   cd gym-website
   # If using python for a local server:
   python -m http.server 8000
   # Then visit http://localhost:8000 in your browser.
   ```
